About & How It Works

Voice folders represent ACNH personality types — each has a unique tone. Audio is streamed live from the GitHub repo.

How text converts: Letter combinations that match available kana clips (for example ba, hyo, iyu) are grouped into one syllable token and play the matching sample. When locale = English/Deutsch and the voice is Bonyari, single letters with dedicated swkbd samples (F G J N R Z in EN; J Y in DE) use the real per-letter pronunciations.

Export WAV stitches samples via Web Audio OfflineAudioContext into one downloadable file.

Voices: aneki (older sister) · bonyari (airhead) · futsu (normal) · genki (peppy) · ghost (spooky) · hakihaki (jock) · kiza (snooty) · kowai (cranky) · otona (smug)
